Information


Authentication System Information

    - This website uses the public Schoolbox API to authenticate and login a user.
    - No credentials are logged or stored, the users session is stored through a temporary cookie.
    - Loggining in through this website should not effect your login status on Schoolbox.

Chatroom Information

    - The chatroom uses websockets to communicate live with the server.
    - All messages are logged, including message content and author.
    - Messages are logged for the purpose of viewable message history. You can view up to 100 previous messages.
    - The content of deleted messages is not logged or stored.
